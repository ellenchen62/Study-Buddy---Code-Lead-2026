Session setup

Focus: 10 min–24 h · Break: 1–60 min · Long break: up to 1.5 h · break is always kept ≤ focus.

Why it works

Short, bounded sprints lower the activation cost of starting and protect attention from fatigue. Pair each focus block with one method — e.g. recall cards, or a Feynman write-up.

How to import into Google Calendar
  1. Click Download .ics above to save the file.
  2. Open Google Calendar → Settings → Import & export.
  3. Choose the .ics file, pick which calendar to add it to, and click Import.
  4. Your study blocks appear as weekly recurring events. (Re-import after you tweak the schedule to refresh.)

Live AI (optional)

The assignment helper works offline with guided templates. To get real AI-written summaries/tips, paste your own API key — it's stored only in this browser and sent straight to the provider.

Why can't it use Claude's key for free?

This page runs entirely in your browser with no server behind it, so there's no shared key it can borrow. Adding your own key (or running it through the templates) are the two safe options. Keys here never leave your machine except to call the provider you chose.

Connecting real Google Classroom later

Step-by-step (when you're ready)
  1. Create a free project at console.cloud.google.com.
  2. Enable the Google Classroom API under APIs & Services.
  3. Configure the OAuth consent screen (External, add yourself as a test user).
  4. Create an OAuth Client ID of type "Web application". Add your hosting URL to Authorized JavaScript origins (Classroom blocks file://, so host on e.g. GitHub Pages/Netlify).
  5. Use scope classroom.coursework.me.readonly with Google Identity Services to fetch assignments.
  6. Paste the Client ID below and press Connect on the Classroom page — the demo data is swapped for your real courses.
